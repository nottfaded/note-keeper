**Note Keeper** is a web application that enables users to create, store, and manage their notes efficiently.

## Technologies Used

- **Frontend**:
  - **TypeScript**
  - **SCSS**
  - **React**
  - **Jotai**
  - **Bun**

- **Backend**:
  - **ASP.NET**
  - **MongoDB**
  - **Google OAuth**

- **Containerization**:
  - **Docker**

- **Development Practices**:
  - **Modular Architecture**
  - **Microservices**
  - **Responsive Design**
  - **RESTful APIs**
